RedirectedStreams¶

class RedirectedStreams(stdout=None, stderr=None)[source]¶

A context manager for capturing stdout and stderr output.

>>> import uqbar.io
>>> import io

>>> string_io = io.StringIO()
>>> with uqbar.io.RedirectedStreams(stdout=string_io):
...     print("hello, world!")
... 

>>> result = string_io.getvalue()
>>> string_io.close()
>>> print(result)
hello, world!

This context manager is not reentrant. Use a separate instance when nesting multiple timers.
