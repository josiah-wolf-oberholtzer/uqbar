A summarizing module documenter.

Organizes member documenters by their documentation section.

Treats nominative submodule documenters (almost) as if they were locally defined members. This means that a documenter for a package which contains a module which contains only one class or function named identically to that module will treat that class or function as though it was defined in the packageâs __init__.py. These nominative submodule members are organized in each documentation section via an autosummary table, rather than including their documentation directly.

>>> import uqbar.apis
>>> documenter = uqbar.apis.SummarizingModuleDocumenter(
...     "uqbar.io",
...     module_documenters=[uqbar.apis.ModuleDocumenter("uqbar.io.Timer"),],
... )
>>> print(str(documenter))
